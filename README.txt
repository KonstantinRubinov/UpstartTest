This is an example for UpstartTest system.

* .NET Core 3 server
* MSSql database
* Angular + Material 9 front - with lazy load.

- All CRUD query for sql database.
- Entity Framework Code First

* Name – required, max length 100 characters
* Amount – number, max 6 characters
* Date – displayed like “21/02/17, But after user decides to save the form it converted into ticks (Long)
